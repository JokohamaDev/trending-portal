# Trending Portal v1.0.1

Discover what's trending in Vietnam across multiple platforms in real-time.

## Overview

Trending Portal is a unified dashboard that brings together the latest trending content from Vietnam's most popular platforms. Whether you're looking for the hottest music, viral videos, trending movies, search topics, or breaking news, you'll find it all in one place.

## What We Track

- **Music**: Top tracks and artists from Spotify Vietnam
- **Videos**: Trending videos from YouTube Vietnam
- **Movies & TV**: Popular content on Netflix Vietnam
- **Search Trends**: What people are searching for on Google
- **News**: Latest headlines from Tuổi Tre Newspaper
- **Gaming**: Top games on Steam

## Features

- **Real-time Updates**: Data refreshes automatically to keep you current
- **Multiple Layouts**: Choose between different viewing styles
- **Dark Mode**: Easy on the eyes, day or night
- **Responsive Design**: Works on desktop, tablet, and mobile
- **Source Health**: See which data sources are currently active

## How to Use

Simply visit the website to see all trending content at a glance. Each section shows:
- Rank position
- Title and relevant details
- Thumbnail images (when available)
- Links to the original source

Use the refresh button to get the latest data, or toggle between light and dark mode for your preferred viewing experience.

## About

Built for anyone who wants to stay informed about Vietnam's digital culture and trending topics. Perfect for content creators, marketers, researchers, or anyone curious about what's popular in Vietnam.
